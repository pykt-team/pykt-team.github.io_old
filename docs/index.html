Welcome to Pykt’s documentation!

PyKT is a python library build upon PyTorch to train deep learning based knowledge tracing models. The library consists of a standardized set of integrated data preprocessing procedures on 7 popular datasets across different domains, 5 detailed prediction scenarios, 10 frequently compared DLKT approaches for transparent and extensive experiments. Let’s Get Started!

Installation

To use pykt, first install it using pip:

$ pip install pykt-toolkit
